# Data Folder

This project uses the Steam dataset provided in the "datasets.zip" package.

Due to size limitations, the raw data is **not included in this repository**.

To run the project:
1. Download the dataset from https://huggingface.co/datasets/sunajcs/steam.
2. Extract all CSV files to:
    data/raw/

Expected files:
- bundles.csv
- extended_games.csv
- games.csv
- item_reviews.csv
- test_interactions_in.csv
- train_interactions.csv
- user_reviews.csv
